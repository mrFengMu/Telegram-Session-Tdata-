# 🚀 Telegram Session/Tdata 转换工具

一个专业的 Telegram 会话格式转换工具，支持 **session** 与 **tdata** 双向转换。**支持少量测试**。

## ✨ 核心功能

### 📤 Session 转 Tdata
- 将  session 文件转换为 tdata 文件夹
- 保留原始会话所有特性与身份认证信息
- 支持多种 session 格式

### 📥 Tdata 转 Session
- 将  tdata 文件夹转换为兼容各种库的 session 文件
- 支持主流框架
- 无缝保留所有授权信息

### 🛡️ 安全特性
- 数据本地处理，不经过第三方服务器
- 保护用户隐私和账号安全
- 不记录敏感信息

## 📊 使用场景

- Telegram 自动化应用开发
- 跨平台账号迁移
- **支持**上传至服务器接码使用
- 批量账号**管理**
